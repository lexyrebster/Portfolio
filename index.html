About Me

October 2020

Hi, my name is Alexis Reb. I'm currently a full-stack development program student through the University of Washington Coding Bootcamp. I enjoy working on the front-end of web development; particularly designing and building functionality to a website.

I think it would be cool to eventually build a website to support my interest in photography, as well as creating an application that allows students to start learning program languages earlier in schools. By making program language learning more accessible to students in early education, it will allow the technology field to be more inclusive and allow individuals to better understand the importance of technology in today's society. I think that adaptation to programming and technology is the way for a successful future.
